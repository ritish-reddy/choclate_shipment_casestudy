# 🍫 Global Chocolate Shipments — Python Analysis

## Project Overview


> Analyzed global shipment data using Excel (data cleaning & pivot analysis), Python (data processing & trend analysis), and Power BI (interactive dashboards) to identify top-selling products and key import markets, enabling insights for pricing strategy, demand forecasting, inventory optimization, and market expansion.

## Data Cleaning Steps (01_data_cleaning.py)

1. **Duplicate removal** — 193 duplicate rows detected and dropped
2. **Date standardisation** — Mixed date formats unified to `datetime64`
3. **Null imputation** — Missing `Amount` and `Boxes` filled with product-level median
4. **Negative value correction** — Sign errors corrected with `abs()`
5. **Order status normalisation** — Typos and variants mapped to standard values
6. **Derived columns** — Year, Month, Quarter, Weekday added
7. **Dimension merges** — Product, geography, and salesperson tables joined
